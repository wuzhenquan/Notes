- 关于`setTimeout()`里的this

  setTimeout(function(){},1000)里面的this指向的值window对象, 要想指向调用setTimeout()的对象

  - 第一种

    ```javascript
    var that = this;
    setInterval(function(){
        return that.baz();
    }, 1
    ```

  - 第二种

    ```javascript
    setInterval( this.baz.bind(this), 1000 );
    ```
